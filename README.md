# Lean-Process-Design-Project
Industrial Management project focused on value creation using lean manufacturing principles and process optimization.
# Value Creation Through Lean Process Design in Manufacturing

## Objective
The objective of this project is to analyze manufacturing waste and improve operational efficiency using lean manufacturing principles.

## Introduction
Modern manufacturing industries face challenges such as high production costs, inventory waste, long lead times, and quality defects. Lean manufacturing helps industries eliminate non-value-added activities and improve customer value.

## Lean Tools Used
- 5S
- Kaizen
- Just-In-Time (JIT)
- Value Stream Mapping
- Kanban
- Poka-Yoke

## Case Study
This project considers a smartphone manufacturing process and analyzes how lean process design improves productivity and reduces waste.

## Manufacturing Flow
Component Arrival
↓
PCB Assembly
↓
Screen Installation
↓
Battery Fitting
↓
Testing
↓
Packaging

## Identified Wastes
- Excess inventory
- Waiting time
- Defects during testing
- Unnecessary worker movement
- Overproduction

## Improvements Suggested
- Implementation of 5S workplace organization
- Better inventory control using JIT
- Improved workstation layout
- Continuous monitoring and Kaizen activities

## Results
| Parameter | Before Lean | After Lean |
|---|---|---|
| Lead Time | 12 hrs | 7 hrs |
| Defect Rate | 8% | 2% |
| Inventory | 500 units | 280 units |
| Productivity | Moderate | Improved |

## Conclusion
Lean manufacturing helps organizations reduce waste, improve operational efficiency, increase customer value, and achieve better productivity.
